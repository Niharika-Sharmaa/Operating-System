🖥️ Operating System Lab Programs 

This repository contains all practical programs implemented for the Operating System Lab.
The programs cover shell scripting, process management, scheduling algorithms, memory management, disk scheduling, and file allocation techniques.

📂 Programs Included - 

🔹 Shell Scripting
marksheet.sh → Generate student marksheet (total, percentage, class)
menu_driven_os.sh → Menu-based script (calendar, date & time, users, terminal)
fibonacci.sh → Generate first n Fibonacci numbers
prime_numbers.sh → Display first n prime numbers

🔹 Process Management
fork_process.c → Demonstrates parent and child process using fork()

🔹 CPU Scheduling
fcfs_scheduling.c / fcfs_scheduling.py → First Come First Serve scheduling
Calculates:
Waiting Time
Turnaround Time
Response Time
CPU Utilization

🔹 Synchronization (IPC)
producer_consumer.c → Producer-Consumer problem using semaphores, mutex, threads

🔹 Deadlock Avoidance
bankers_algorithm.c → Banker’s Algorithm (safe state & safe sequence)

🔹 Memory Management
fifo_page_replacement.c → FIFO page replacement
Calculates:
Page Faults
Page Hits
Hit Ratio
Miss Ratio

🔹 Disk Scheduling
disk_fcfs.c → FCFS disk scheduling (total head movement)

🔹 File Management
file_allocation.c →
Contiguous Allocation
Linked Allocation
Indexed Allocation

📌 Key Concepts Covered
-Process Creation (fork)
-CPU Scheduling (FCFS)
-Synchronization (Producer–Consumer)
-Deadlock Avoidance (Banker’s Algorithm)
-Page Replacement (FIFO)
-Disk Scheduling (FCFS)
-File Allocation Methods
-Shell Scripting Basics

🎯 Objective
To understand and implement core Operating System concepts through practical programs, including process management, scheduling, memory management, file systems, and inter-process communication.
